# AGENTS.md

## Project

Norgesgym is a local gym at Solgaard Skog in Moss, Norway. The website should position Norgesgym as a modern, local, inclusive strength gym.

## Design Direction

Use the light, spacious homepage direction. The site should feel clean, calm, confident, local, and welcoming to women, beginners, and adults, while still credible as a strength-training gym.

## Core Visual Principles

- Light / warm-white layout
- Large editorial typography
- Generous spacing
- Right-side hero image card
- Purple as a controlled signal color
- Minimal icon usage
- Clear CTA path
- Real website viewport feel, not a poster

## Avoid

- Hardcore bodybuilding aesthetic
- Black/red gym branding
- Norwegian flag colors as main identity
- Aggressive no-pain-no-gain language
- Dense poster-like layouts
- Too many cards, icons, or badges
- Generic AI fitness templates
- Dark full-page landing-page look

## Preferred Tokens

- `ink`: `#0D1117`
- `muted`: `#5F6673`
- `background`: `#FFFFFF`
- `warm background`: `#F7F5F1`
- `surface`: `#FFFFFF`
- `border`: `#E5E2DC`
- `primary purple`: `#6D35D3`
- `purple dark`: `#3B1A78`
- `purple soft`: `#EEE7FF`
- `night`: `#090D14`

## Typography

- Use Manrope for headings.
- Use Inter for body text.

## Implementation

- Use Next.js, TypeScript, and Tailwind CSS.
- Prefer clean reusable components.
- Keep site content and external links in `/lib/site-config.ts`.
- Avoid overengineering.
- Do not add many new sections unless explicitly asked.

## Quality Bar

Before finishing a task, check that the page still feels light, spacious, and modern, not like an information poster or generic gym template.
